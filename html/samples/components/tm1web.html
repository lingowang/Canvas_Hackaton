TM1 Web

Criteria
Cube View
Use tm1-cube and tm1-cube-view to utilize TM1's Cube Viewer
You can update the title in your Cube View via tm1-titles
Web Sheet
Use tm1-application to utilize TM1's Websheet Viewer
If it has a SUBNM, you can update it too via the tm1-titles attribute.
Cube View with Toolbar Shown
Use tm1-show-tool-bar to show the standard TM1 tool bar.
Web Sheet with Toolbar Shown
You can use tm1-show-tool-bar in here too.
Width within its Container
Use tm1-width to assign it with a more specific width relative to its container.
Control the Height
Use tm1-height to control the height it will take when shown.
Apply a CSS style on the Component

Use tm1-ui-class to affect the style of the component further.
