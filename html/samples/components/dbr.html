DBR - Simple
Use the DBR element to retrieve a value from a cube
Use must supply the tm1-instance, tm1-cube and tm1-elements attributes
DBR - Decimal Places
To specify the number of decimal places use tm1-data-decimal="0" and tm1-read-only="true" to set to read only
DBR Percentage (%)
To format the number as a percentage add the attribute tm1-format-percentage="true"
DBR (with Validation)
Validate entry on input boxes. Put your expression to evaluate in tm1-validate.
Show helpful hints when validation fails via tm1-validate-message.
DBR - Comments for Cube Input Cells

DBR with Picklist on Cell

Setting the dbr to read only tm1-read-only="true" will remove the drop down.
DBR with Multi Line Support!
