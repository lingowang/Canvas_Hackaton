List of Elements Combined with ng-class

Region Alias
{{item.alias}}
Use tm1-ui-class to affect the style of the component further.
For more advance formatting, use with Angular's ng-class to style the row ng-class="item.topLevelClass".
<tr ng-repeat="item in page.list6 | filter:page.search6" ng-class="item.topLevelClass">...</tr>
This property of the the current item exposes a CSS class which you can then use to customize the the row display, depending on its level.
Each row will have been added a class tm1-ui-level-* corresponding to N if it is a Leaf element (i.e. tm1-ui-level-N) or a number, corresponding to whether it is a top level (i.e. tm1-ui-level-0) or one level below (i.e. tm1-ui-level-1) and so on.
.tm1-ui-level-N{ color: #162856; } .tm1-ui-level-0{ color: #4F618F; font-weight: bold; font-size: 20px; } .tm1-ui-level-1{ color: #4F618F; font-weight: bold; font-size: 16px; } .tm1-ui-level-2{ color: #4F618F; font-weight: bold; font-size: 14px; }
