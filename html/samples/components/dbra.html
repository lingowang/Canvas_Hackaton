DBRA

DBRA (Input)
Set the tm1-read-only attribute to false (default) to make it writable
DBRA (Read Only)
Set the tm1-read-only attribute to true to make the DBRA read only
DBRA (Input with Custom Empty Display)
Set the tm1-placeholder attribute to set the background text when empty.
DBRA (Decimal Format)
Set 2 decimals format using tm1-data-decimal when in read-only mode via tm1-read-only
DBRA (Validation)
Validate entry on input boxes. Put your expression to evaluate in tm1-validate.
Show helpful hints when validation fails via tm1-validate-message.
DBRA (Hide Cell Reference)
Hide the context button using tm1-hide-reference
DBRA (tm1-api)
{{values.dbraDisplayValue}}
